# TeBuscan 🔍

Portal web de búsqueda de empleo que conecta candidatos con empresas. Proyecto universitario desarrollado con Node.js, Express y Supabase.

---

## 🛠️ Tecnologías

- **Backend:** Node.js, Express
- **Base de datos:** Supabase (PostgreSQL)
- **Frontend:** HTML, CSS, JavaScript vanilla, Bootstrap 5.3

---

## 📋 Descripción

TeBuscan permite a los candidatos explorar ofertas de trabajo, postularse a posiciones y valorar empresas. Las empresas pueden publicar empleos y gestionar sus candidatos. La plataforma cuenta con tres roles: público, candidato/empresa y administrador.

---

## ⚙️ Instalación

```bash
git clone https://github.com/tu-usuario/tebuscan.git
cd tebuscan
npm install
npm run dev
```

### Variables de entorno

```env
SUPABASE_URL=https://xxxx.supabase.co
SUPABASE_KEY=your_anon_key
PORT=3000
```
